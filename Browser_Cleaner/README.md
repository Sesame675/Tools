
# Browser Data Cleaner
A lightweight Chrome (Manifest V3) extension that identifies and removes residual
browser storage associated primarily with third-party domains, such as trackers,
analytics services, and embedded resources, while preserving data for frequently
visited first-party sites.

## Installation (Developer Mode)
1. Clone this repository
2. Open Chrome and navigate to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the project directory
